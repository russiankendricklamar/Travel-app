**1. Add NavigationHUDView overlay in ZStack** — insert AFTER the search pill block and BEFORE the bottom sheet block (between the `// MARK: - Search Pill` section and `// MARK: - Bottom Sheet` section):

```swift
// MARK: - Navigation HUD (floating top card)
if vm.isNavigating {
    VStack {
        NavigationHUDView(vm: vm)
            .padding(.horizontal, 16)
            .padding(.top, 8)
        Spacer()
    }
    .transition(.move(edge: .top).combined(with: .opacity))
    .animation(.easeInOut, value: vm.isNavigating)
}
```

**2. Add MapRecenterButton in ZStack** — insert directly after the HUD block:

```swift
// MARK: - Recenter Button (appears on manual pan during navigation)
if vm.isNavigating && vm.isOffNavCenter {
    VStack {
        Spacer()
        MapRecenterButton {
            vm.recenterNavigation()
        }
        .padding(.bottom, 100) // Above the bottom sheet peek
    }
    .animation(.spring(response: 0.3), value: vm.isOffNavCenter)
}
```

**3. Add `.navigation` case to sheetBody** — in the `switch vm.sheetContent` block, add after `case .routeInfo:`:

```swift
case .navigation:
    NavigationSheetContent(vm: vm)
```

**4. Guard dismissDetail() call in onChange(of: vm.sheetDetent)** — modify the existing handler from:
```swift
.onChange(of: vm.sheetDetent) { oldDetent, newDetent in
    if newDetent == .peek && oldDetent != .peek {
        vm.dismissDetail()
    }
}
```
To:
```swift
.onChange(of: vm.sheetDetent) { oldDetent, newDetent in
    if newDetent == .peek && oldDetent != .peek && !vm.isNavigating {
        vm.dismissDetail()
    }
}
```
This prevents navigation from being interrupted when the user collapses the sheet to peek.

**5. Add manual pan detection in onMapCameraChange** — modify the existing `.onMapCameraChange` callback in `mapContent` from:
```swift
.onMapCameraChange { context in
    vm.visibleRegion = context.region
}
```
To:
```swift
.onMapCameraChange { context in
    vm.visibleRegion = context.region
    // Detect manual pan during navigation — show recenter button if > 50m away
    if vm.isNavigating, let userLoc = LocationManager.shared.currentLocation {
        let center = context.region.center
        let distance = CLLocation(latitude: center.latitude, longitude: center.longitude)
            .distance(from: CLLocation(latitude: userLoc.latitude, longitude: userLoc.longitude))
        vm.isOffNavCenter = distance > 50
    }
}
```

**Also update `isIdleMode`** — the existing computed property checks `vm.sheetDetent == .peek && vm.sheetContent == .idle`. This is correct: when navigating, sheetContent is .navigation (not .idle), so isIdleMode is false and the sheet is shown. No change needed.

**Also: Update bottom sheet visibility condition** — the current condition is `!isOfflineWithCache && !isIdleMode`. This is correct because during navigation sheetContent==.navigation which means isIdleMode==false, so the sheet shows. No change needed.
