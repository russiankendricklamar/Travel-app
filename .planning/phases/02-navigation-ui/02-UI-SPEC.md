## Spacing Scale

All tokens match existing `AppTheme` constants. Use these token names in code.

| Token | Value | AppTheme constant | Usage |
|-------|-------|-------------------|-------|
| xs | 4pt | `AppTheme.spacingXS` | Icon gaps, accent bar widths |
| sm | 8pt | `AppTheme.spacingS` | Compact element spacing, internal VStack gaps |
| md | 16pt | `AppTheme.spacingM` | Default element padding, horizontal card padding |
| lg | 24pt | `AppTheme.spacingL` | Section padding, sheet content padding |
| xl | 32pt | `AppTheme.spacingXL` | Layout gaps, bottom padding in sheets |

**Exceptions:**

| Context | Value | Reason |
|---------|-------|--------|
| HUD card horizontal padding | 12pt | Tighter than md — preserves compact HUD width |
| HUD card vertical padding | 12pt | Tighter than md — keeps HUD height ≤ 80pt |
| HUD direction icon container | 52×52pt | Touch-safe icon target for urgency zone |
| MapRecenterButton horizontal padding | 12pt | Matches HUD internal rhythm |
| MapRecenterButton vertical padding | 8pt | Capsule pill height constraint |
| "НАЧАТЬ НАВИГАЦИЮ" button vertical padding | 16pt | Standard tap target height |
| Navigation sheet peek row horizontal padding | 16pt | Matches sheet shell |
| Navigation sheet peek row vertical padding | 12pt | Compact strip height |
| Safe area inset top (HUD positioning) | 8pt | Gap between Dynamic Island / notch and HUD card |

## Typography

All sizes use `.system(size:weight:)` — no custom font loading required.

**Weights:** exactly 2 — regular (400) and semibold (600).

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| HUD instruction | 15pt | semibold (600) | 1.3 (lineLimit 2) | Primary maneuver text in HUD card |
| HUD distance | 13pt | regular (400) | 1.0 | Distance below instruction in HUD |
| Sheet peek instruction | 14pt | semibold (600) | 1.0 (lineLimit 1) | Current step in collapsed sheet |
| Sheet peek secondary | 12pt | regular (400) | 1.0 | Trip context label + "прибытие" label + sheet peek ETA value |
| Step list instruction | 14pt | regular (400) | 1.4 | Individual steps in expanded step list |
| Step list current | 14pt | semibold (600) | 1.4 | Highlighted current step in list |
| Step list distance | 12pt | regular (400) | 1.0 | Per-step distance in step list |
| Start button label | 15pt | semibold (600) | 1.0 | "НАЧАТЬ НАВИГАЦИЮ" button text |

**Tracking exceptions:**
- "НАЧАТЬ НАВИГАЦИЮ" button text: `tracking(1)` — letter-spaced uppercase
- Section headers: `tracking(2)` — matches `GlassSectionHeader` pattern (use 12pt regular for any section label caps)

**Design note:** Exactly 4 distinct size roles (15, 14, 13, 12pt) and exactly 2 weight roles (regular 400, semibold 600). Consistent with the project's existing typography approach.

## Color

All color values are palette-aware via `AppTheme.sakuraPink` (reads `ColorPalette.current.accentColor` at runtime). Do not hardcode hex values for accent — always reference the `AppTheme` token.

| Role | Token / Value | Usage |
|------|---------------|-------|
| Dominant (60%) | `.ultraThinMaterial` (system adaptive) | HUD card background, navigation sheet background |
| Secondary (30%) | `.primary` / `.secondary` (system adaptive) | Text, icons in normal state |
| Accent (10%) | `AppTheme.sakuraPink` | Reserved-for list below |
| Destructive | `AppTheme.toriiRed` (`#E11D48`) | Not used in this phase — no destructive actions |

**Accent (`AppTheme.sakuraPink`) reserved for:**
1. HUD direction icon foreground (normal state)
2. HUD direction icon container fill at urgency state (distance < 50m)
3. HUD distance text foreground at urgency state (distance < 50m)
4. "НАЧАТЬ НАВИГАЦИЮ" button fill (solid, full-width)
5. MapRecenterButton capsule fill
6. MapRecenterButton shadow tint (`AppTheme.sakuraPink.opacity(0.4)`)
7. Navigation sheet peek ETA value text
8. Navigation sheet expanded step list — current step icon foreground
9. Navigation sheet expanded step list — current step accent bar (left edge)
10. AccentCardStyle gradient stroke on HUD (when urgency active, swap to AccentCardStyle)

**Urgency color transition contract:**

| State | Icon container fill | Icon foreground | Distance text |
|-------|--------------------|-----------------|----|
| Normal (≥ 65m) | `AppTheme.sakuraPink.opacity(0.15)` | `AppTheme.sakuraPink` | `.secondary` |
| Urgent (< 50m) | `AppTheme.sakuraPink` (solid) | `.white` | `AppTheme.sakuraPink` |

Hysteresis: activate urgency at < 50m, deactivate at > 65m (prevents GPS noise flicker). Source: RESEARCH.md Pitfall 4.

**Glass border colors:**
- Light scheme: `Color.white.opacity(0.3)` — 0.5pt stroke
- Dark scheme: `Color.white.opacity(0.12)` — 0.5pt stroke

## Interaction States

### HUD Card (`NavigationHUDView`)

| State | Trigger | Visual |
|-------|---------|--------|
| Hidden | `vm.isNavigating == false` | Not rendered; transition: `.move(edge: .top).combined(with: .opacity)` |
| Visible — normal | `vm.isNavigating == true`, distance ≥ 65m | `CardStyle` glass card; icon container `sakuraPink.opacity(0.15)` |
| Visible — urgent | `vm.isNavigating == true`, distance < 50m | Icon container solid `sakuraPink`; icon white; distance text `sakuraPink` |
| Step change | `vm.currentStepIndex` increments | Icon: `.contentTransition(.symbolEffect(.replace))`; text: `.contentTransition(.opacity)`; animation: `.easeInOut(duration: 0.3)` |

### "НАЧАТЬ НАВИГАЦИЮ" Button

| State | Visual |
|-------|--------|
| Default | Solid `AppTheme.sakuraPink` fill, `.white` text, `AppTheme.radiusMedium` (16pt) corner radius |
| Disabled | `vm.navigationSteps.isEmpty && vm.isCalculatingRoute` → `.disabled(true)` (system opacity reduction) |

### MapRecenterButton

| State | Trigger | Visual |
|-------|---------|--------|
| Hidden | `!vm.isNavigating \|\| !vm.isOffNavCenter` | Not rendered; transition: `.scale.combined(with: .opacity)` |
| Visible | `vm.isNavigating && vm.isOffNavCenter` | Sakura pink capsule, shadow `sakuraPink.opacity(0.4)` radius 8 y 4 |

Recenter threshold: camera center diverges > 50m from user location during active navigation.

### Navigation Sheet (`NavigationSheetContent`)

| Detent | Content |
|--------|---------|
| `.peek` | Compact strip: direction icon + current instruction (lineLimit 1) + ETA right-aligned + trip context label below instruction |
| `.half` / `.full` | Scrollable `NavigationStep` list; current step highlighted with sakuraPink accent bar + semibold text; "Завершить навигацию" button at bottom |

Sheet dismiss guard: `onChange(of: vm.sheetDetent)` must NOT call `vm.dismissDetail()` when `vm.isNavigating == true`. Source: RESEARCH.md Pitfall 3.

### Camera Transitions

| Event | Camera Behavior | Animation |
|-------|----------------|-----------|
| `startNavigation()` | `cameraPosition = .userLocation(followsHeading: true, fallback: .automatic)` | `withAnimation(.easeInOut(duration: 0.8))` |
| `stopNavigation()` | `cameraPosition = .automatic` | `withAnimation(.easeInOut(duration: 0.5))` |
| Manual pan detected | `vm.isOffNavCenter = true` → recenter button appears | Button: `.scale.combined(with: .opacity)` |
| Recenter tapped | `cameraPosition = .userLocation(followsHeading: true, fallback: .automatic)` | `withAnimation(.easeInOut(duration: 0.5))` |

## SF Symbol Mapping

Direction icons parsed from `MKRouteStep.instructions` string. Always reference `NavigationHUDView.iconForInstruction(_:)` — single source of truth for both HUD and sheet.

| Keyword match (lowercased) | SF Symbol |
|---------------------------|-----------|
| "налево" / "left" | `arrow.turn.up.left` |
| "направо" / "right" | `arrow.turn.up.right` |
| "разворот" / "u-turn" | `arrow.uturn.left` |
| "слег" + "лев" | `arrow.up.left` |
| "слег" + "прав" | `arrow.up.right` |
| "прямо" / "straight" / "continue" | `arrow.up` |
| "пункт" / "прибыт" / "destination" | `mappin.circle.fill` |
| default (no match) | `arrow.up` |

Dismiss button (HUD X): `xmark.circle.fill` — `.symbolRenderingMode(.hierarchical)`, `.secondary` foreground. Apply `.accessibilityLabel("Завершить навигацию")` — icon-only control requires explicit VoiceOver label.
Recenter button icon: `location.fill` — 13pt semibold, `.white`.
Start button icon: `location.fill` — 14pt semibold, `.white`.

## Copywriting Contract

All copy is in Russian (project UI language — source: MEMORY.md).

| Element | Copy |
|---------|------|
| Primary CTA — start navigation | **"НАЧАТЬ НАВИГАЦИЮ"** (uppercase, tracking 1) |
| Stop navigation — sheet button | **"Завершить навигацию"** |
| Stop navigation — HUD X | Icon only (`xmark.circle.fill`); `.accessibilityLabel("Завершить навигацию")` required |
| Recenter button | **"Вернуться"** |
| Trip context label format | **"День {N} из {M} — {Город}"** (e.g. "День 2 из 7 — Токио") |
| Sheet peek ETA label | **"прибытие"** (lowercase, below ETA time) |
| Empty state — no steps | Button disabled (no empty state message needed; steps load before navigation starts) |
| Error state — navigation failed | Not in scope for Phase 2 (NavigationEngine handles reroute internally) |
| Destructive confirmation — stop navigation | **None** — single tap stops navigation without confirmation (source: CONTEXT.md locked decision) |

**Destructive actions in this phase:** Stop navigation (single tap, no confirmation). This is intentional per locked decision — navigation UX must not add friction to stopping.

## Animation Contract

| Animation | Type | Duration | API |
|-----------|------|----------|-----|
| HUD appear / disappear | `.move(edge: .top).combined(with: .opacity)` | system default | `.transition()` |
| HUD step icon change | SF Symbol morph | 0.3s | `.contentTransition(.symbolEffect(.replace))` |
| HUD instruction text change | Opacity crossfade | 0.3s | `.contentTransition(.opacity)` |
| HUD urgency color change | Implicit | 0.3s | `.animation(.easeInOut, value: vm.isUrgent)` |
| Camera lock on start | Eased move | 0.8s | `withAnimation(.easeInOut(duration: 0.8))` |
| Camera restore on stop | Eased move | 0.5s | `withAnimation(.easeInOut(duration: 0.5))` |
| Recenter button appear | Scale + opacity | system default | `.transition(.scale.combined(with: .opacity))` |
| Recenter camera snap | Eased move | 0.5s | `withAnimation(.easeInOut(duration: 0.5))` |
| Sheet content switch to .navigation | Spring | response 0.3s | `withAnimation(.spring(response: 0.3))` |
| Sheet content switch to .routeInfo on stop | Spring | response 0.3s | `withAnimation(.spring(response: 0.3))` |

## Layout Contract

### NavigationHUDView layout

```
[HUD card — CardStyle]
  HStack spacing: 12
  ├── ZStack (52×52pt)
  │   ├── RoundedRectangle(cornerRadius: 12) — urgency fill
  │   └── Image(systemName: maneuverIcon) — 24pt semibold
  ├── VStack alignment: .leading spacing: 4 (maxWidth: .infinity)
  │   ├── Text(instruction) — 15pt semibold, lineLimit 2
  │   └── Text(distance) — 13pt regular rounded
  └── Button (xmark.circle.fill) — 22pt hierarchical
      .accessibilityLabel("Завершить навигацию")
  padding: horizontal 12, vertical 12
```

Position in ZStack: Between map and bottom sheet. Pinned to top via:
```swift
VStack {
    NavigationHUDView(vm: vm)
        .padding(.horizontal, 16)
        .padding(.top, 8)
    Spacer()
}
```

### NavigationSheetContent peek layout

```
VStack spacing: 0
  HStack spacing: 8
  ├── Image(systemName: icon) — 18pt semibold, sakuraPink, frame 32×32
  ├── VStack alignment: .leading spacing: 4 (maxWidth: .infinity)
  │   ├── Text(instruction) — 14pt semibold, lineLimit 1
  │   └── Text(tripContextLabel) — 12pt regular, .secondary
  └── VStack alignment: .trailing spacing: 4
      ├── Text(etaString) — 14pt semibold rounded, sakuraPink
      └── Text("прибытие") — 12pt regular, .secondary
  padding: horizontal 16, vertical 12
```

### NavigationSheetContent expanded step list

```
VStack spacing: 0
  ScrollView
    LazyVStack spacing: 0
      ForEach(vm.navigationSteps)
        StepRow
          HStack spacing: 12
          ├── [current step] AccentBar 4×32pt sakuraPink
          │   OR
          │   [other step] Spacer 4×32pt .clear
          ├── Image(systemName: icon) — 16pt, sakuraPink if current else .secondary
          ├── VStack alignment: .leading spacing: 4
          │   ├── Text(instruction) — 14pt semibold if current else regular
          │   └── Text(distance) — 12pt regular, .secondary
          padding: horizontal 16, vertical 10
  Divider
  Button("Завершить навигацию")
    frame: maxWidth .infinity
    padding: vertical 16, horizontal 16
    background: AppTheme.sakuraPink
    foreground: .white
    cornerRadius: AppTheme.radiusMedium
    padding bottom: 24
```

### MapRecenterButton layout

```
Button
  HStack spacing: 8
  ├── Image("location.fill") — 13pt semibold
  └── Text("Вернуться") — 13pt semibold
  foreground: .white
  padding: horizontal 12, vertical 8
  background: Capsule — AppTheme.sakuraPink fill
  shadow: sakuraPink.opacity(0.4), radius 8, y 4
```

Position in ZStack: Below HUD, above sheet. Bottom-center alignment.
