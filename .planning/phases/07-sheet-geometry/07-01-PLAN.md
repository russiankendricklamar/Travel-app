<objective>
Refactor MapBottomSheet.swift to implement correct peek/half/full geometry with Apple Maps-quality materials and unified shape morph.

Purpose: The current sheet uses dual shape types (RoundedRectangle for peek, UnevenRoundedRectangle for expanded) causing shape snap on transition, wrong peek height (50pt instead of 56pt), hardcoded Color.black.opacity(0.75) instead of ultraThinMaterial, wrong handle dimensions, and handle hidden in peek mode. This plan fixes all geometry issues to match Apple Maps.

Output: Refactored MapBottomSheet.swift with correct detent heights, single UnevenRoundedRectangle for all states, ultraThinMaterial peek background, drag handle in all states, and proper safe area handling. Dead MapFloatingSearchPill.swift deleted.
</objective>
