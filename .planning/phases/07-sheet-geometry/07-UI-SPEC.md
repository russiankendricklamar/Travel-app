## Color

Phase 7 operates entirely in dark color scheme (enforced by `.environment(\.colorScheme, .dark)` on the map). All color values are SwiftUI system materials or explicit dark-mode values.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `Color(uiColor: .systemBackground)` in dark scheme (~#1C1C1E) | Half/full expanded sheet background (D-11) |
| Secondary (30%) | `.ultraThinMaterial` + `Color.black.opacity(0.35)` overlay | Peek pill background — allows map tiles to show through (D-10) |
| Accent (10%) | `AppTheme.sakuraPink` (palette-aware) | Reserved: not used in Phase 7 geometry layer |
| Destructive | not used | No destructive actions in this phase |

**Accent reserved for:** Nothing in Phase 7. The drag handle uses `Color(.systemFill)` — a neutral system gray. Accent color (sakuraPink) does NOT appear in the sheet geometry layer.

### Shadow Contracts (from D-12, D-13)

| Element | Shadow spec |
|---------|------------|
| Peek pill | `shadow(color: .black.opacity(0.35), radius: 14, x: 0, y: 4)` |
| Expanded sheet (top edge) | `shadow(color: .black.opacity(0.15), radius: 10, y: -5)` |

### Background Transition (D-14)

Peek → half background morph: opacity crossfade via `.transition(.opacity)`, duration 0.15s, `easeInOut` curve. NOT a positional animation — the two background layers crossfade.

## Interaction Contracts

### Drag Gesture

| Property | Value | Source |
|----------|-------|--------|
| Minimum distance | 5pt | D-34 |
| Velocity threshold | 0.3 (relative to screen height) | D-30 |
| Velocity probing | 20% screen height lookahead on fling | D-30 |
| Spring | `response: 0.35, dampingFraction: 0.85` — FIXED, do not adjust | D-31 |
| Detent calculation | `SheetDetent.nearest(to:in:)` — nearest by absolute height distance | existing code |

### Peek → Half/Full Tap

Tapping the pill in peek mode expands to full and focuses search (Phase 8). Phase 7 only establishes the geometry — tap target is the entire 56pt pill surface.

### Tab Bar Integration (D-35, D-07)

- In peek/idle mode: tab bar visible, pill floats above it
- In half/full mode: tab bar hidden via `.toolbar(.hidden, for: .tabBar)`
- Map bottom safe area: `.safeAreaPadding(.bottom, isIdleMode ? 66 : 0)` on the map view

### Keyboard (D-29)

Sheet uses `.ignoresSafeArea(.keyboard)` — it manages its own vertical position when keyboard appears. No auto-scroll or layout shift from keyboard.
