Create a new file `PolylineSmoother.swift` in `Travel app/Views/Map/` with an enum `PolylineSmoother` containing two static methods:

1. **`simplify(coordinates:epsilon:) -> [CLLocationCoordinate2D]`** — Douglas-Peucker algorithm.
   - `epsilon` parameter in degrees (default 0.00005, roughly 5 meters).
   - Reduces GPS noise by removing points that deviate less than epsilon from the simplified line.
   - Guard: if fewer than 3 points, return as-is.
   - Use perpendicular distance from point to line segment (standard DP formula adapted for lat/lon — fine to treat as flat at city scale).

2. **`smooth(coordinates:pointsPerSegment:epsilon:) -> [CLLocationCoordinate2D]`** — Pipeline: simplify first, then Catmull-Rom spline interpolation.
   - `pointsPerSegment` default 8 (number of interpolated points between each pair of simplified points).
   - Catmull-Rom: for each segment i, use control points [i-1, i, i+1, i+2]. For first/last segments, duplicate the endpoint as the missing control point (standard clamped Catmull-Rom).
   - Formula per axis: `0.5 * ((2*p1) + (-p0+p2)*t + (2*p0-5*p1+4*p2-p3)*t^2 + (-p0+3*p1-3*p2+p3)*t^3)` where t goes from 0 to 1 in `pointsPerSegment` steps.
   - Guard: if fewer than 2 points after simplification, return simplified points.
   - Return the full interpolated coordinate array including original (simplified) points.

Import Foundation and MapKit. No SwiftUI import needed.
