## What Was Built

### MapSearchContent.swift
- `todayPlacesSection`: visible in sheet idle+focused state when `vm.trip.todayDay` exists. Shows section header with city name and "visitedCount/total" counter. Each place row has category icon in circle, name, and visited checkmark (`checkmark.circle.fill` in bambooGreen or `circle` in tertiary). Tapping sets `vm.selectedPlaceID`, dismisses keyboard, collapses sheet to peek.
- `mapControlsSection`: horizontal scrollable row of 4 icon+label buttons (Слои, Осадки, Обзор, Все места). Layers button is a Menu with all layer toggles. Precipitation button tints to `AppTheme.oceanBlue` when active.
- `mapControlButton()`: private helper building the 36x36 circle + label VStack.

### TripMapView.swift
- Replaced `.toolbarBackground(.hidden)` + `.toolbarColorScheme(.dark)` + `.toolbar { toolbarContent }` with `.toolbar(.hidden, for: .navigationBar)`.
- Added floating precipitation capsule overlay in ZStack (`.ultraThinMaterial` Capsule, `.padding(.top, 60)`).
- Deleted `toolbarContent` computed property entirely (93 lines removed).
