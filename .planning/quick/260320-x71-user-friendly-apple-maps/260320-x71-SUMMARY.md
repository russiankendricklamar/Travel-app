## What Was Built

### Task 1 — Materials + Controls
- **MapBottomSheet**: `Color.black` background replaced with `UnevenRoundedRectangle.fill(.ultraThinMaterial)` + top shadow. Native iOS blur now matches Apple Maps sheet behavior.
- **MapFloatingSearchPill**: `Capsule().fill(Color.black)` replaced with `RoundedRectangle(cornerRadius: 12).fill(.ultraThinMaterial)` + shadow. Matches Apple Maps' search bar shape.
- **TripMapView `.mapControls`**: Added `MapCompass()` (auto-hides when north-aligned), `MapUserLocationButton()` (cycles tracking states), `MapPitchToggle()` (enables 3D tilt). All render on trailing edge.

### Task 2 — Category Chips + Look Around
- **MapViewModel**: Added `quickCategories` static array (6 categories: Рестораны/Кафе/Музеи/Парки/Магазины/Отели), `selectedCategory`, `isLoadingCategory`, `performCategorySearch()` method, `lookAroundScene`, `isLoadingLookAround`, `fetchLookAround()` method. Resets applied in `dismissDetail()` and `onPlaceSelected()`.
- **MapSearchContent**: Horizontal `ScrollView` with `Capsule`-shaped chips below search bar. Accent fill (`AppTheme.sakuraPink`) when category active. Loading spinner inline for active category. Visible only in `.idle` and `.searchResults` sheet states.
- **MapPlaceDetailContent**: `lookAroundSection` `@ViewBuilder` renders `LookAroundPreview(initialScene:)` at 180pt height with rounded corners, or a skeleton placeholder while loading. Inserted in both `placeDetailView` and `searchItemDetailView` after photo carousel.
- **TripMapView**: `fetchLookAround` called at end of `task(id: selectedPlaceID)` block; `onChange(of: vm.searchedItem)` triggers fetch for search results.
