### Issue: TRAN-01 Spring Divergence in View Layer

`MapViewModel.sheetSpring` is fully unified (13 call sites). However:
- `TripMapView.swift` line 248: detent mutation uses `.spring(response: 0.3, dampingFraction: 0.85)` — should reference `sheetSpring`
- `MapSearchContent.swift` lines 42, 73, 291: decorative animations use `.spring(response: 0.3)` — non-detent, lower priority

**Impact:** One detent transition path has slightly different spring timing. Decorative animations are independent.
